Piekarnia sprzedaje pieczywo pełnoziarniste, słodycze i ciasta. Aby zachęcić do sprzedaży, dyrektor ogłosił nagrody w oparciu o liczbę produktów sprzedanych w ciągu tygodnia, zgodnie z systemem punktowym: każdy bochenek chleba (b) jest wart 1 punkt, każda paczka słodyczy (s) jest warta 2 punkty, a każde ciasto (c) jest warte 3 punkty. W zależności od sumy zgromadzonych punktów, pracownicy mogą otrzymać inną nagrodę. Wyzwanie polega na stworzeniu programu, który określa, jaką nagrodę otrzymają pracownicy na podstawie ilości sprzedanych produktów.

Jeśli suma punktów wszystkich produktów sprzedanych w danym tygodniu jest równa lub większa niż 150, każdy pracownik otrzymuje ciasto ("C") jako nagrodę; w przeciwnym razie, jeśli suma punktów jest większa lub równa 120, każdy pracownik otrzymuje paczkę słodyczy ("S") jako nagrodę; w przeciwnym razie, jeśli suma punktów jest większa lub równa 100, każdy pracownik otrzymuje bochenek chleba ("B") jako nagrodę. Jeśli suma punktów jest mniejsza niż 100, pracownicy nie otrzymują żadnej nagrody ("N").

Wyzwanie polega na stworzeniu programu, który określa, jaką nagrodę otrzymają pracownicy na podstawie ilości każdego sprzedanego produktu.

Dane wejściowe: Trzy liczby całkowite ({% if is_js %}number{% else %}int{% endif %}) reprezentujące odpowiednio liczbę sprzedanych w tygodniu chlebów, paczek słodyczy i ciastek.

Wyjście: Ciąg znaków (str{% if is_js %}ing{% endif %}) zawierający znak reprezentujący nagrodę.
