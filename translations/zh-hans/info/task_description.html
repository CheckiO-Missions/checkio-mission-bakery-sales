一家面包店销售全麦面包、糖果和蛋糕。为了鼓励销售，店长宣布根据一周内售出的产品数量进行奖励，采用积分制：每块面包 (b) 得 1 分，每包糖果 (s) 得 2 分，每个蛋糕 (c) 得 3 分。根据累积的总积分，员工可以获得不同的奖励。挑战在于创建一个程序，根据每种产品的销售数量来决定员工将获得的奖励。

如果本周售出的所有产品的积分之和等于或大于 150，则每位员工可获得一个蛋糕 ("C") 作为奖励；否则，如果积分之和大于或等于 120，则每位员工可获得一个糖果包 ("S") 作为奖励；否则，如果积分之和大于或等于 100，则每位员工可获得一个面包 ("B") 作为奖励。如果积分之和小于 100，则员工没有任何奖励 ("N")。

我们面临的挑战是创建一个程序，根据每种产品的销售数量来决定员工将获得哪种奖励。

输入三个整数 ({% if is_js %}number{% else %}int{% endif %})分别代表一周内售出的面包、糖果包和蛋糕的数量。

输出： 字符串字符串 (str{% if is_js %}ing{% endif %})包含一个代表奖品的字符。
